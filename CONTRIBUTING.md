# Contributing to Vibe AIGC

Thank you for your interest in contributing! See our [full contributing guide](https://jmanhype.github.io/vibe-aigc/contributing/) for detailed instructions.

## Quick Start

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/vibe-aigc.git
cd vibe-aigc

# Install dev dependencies
pip install -e ".[dev]"

# Run tests
pytest -v

# Check code quality
black . && ruff check . && mypy vibe_aigc
```

## Pull Request Process

1. Fork the repo and create your branch from `main`
2. Add tests for any new functionality
3. Ensure all tests pass
4. Update documentation as needed
5. Submit your PR

## Commit Messages

Use conventional commits: `feat:`, `fix:`, `docs:`, `test:`, `refactor:`, `chore:`

## Questions?

Open an issue or start a discussion on GitHub.
